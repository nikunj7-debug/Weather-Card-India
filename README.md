# Weather India Pro 🌤️

Weather India Pro is a responsive weather web application that allows users to search Indian states and cities to view real-time weather information.

## Features

* User Login & Sign Up
* State Search with Suggestions
* Top 10 Major Cities Display
* Additional City Search Option
* Smart Autocorrect & Fuzzy Search
* Real-Time Weather Data
* Rain Prediction
* Humidity Information
* Daily Minimum & Maximum Temperature
* Home Navigation & Logout
* Mobile-Friendly Responsive Design
* Green & Yellow Weather-Themed Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* OpenWeatherMap API
* Local Storage

## Setup

1. Download the project files.
2. Open `index.html`.
3. Replace the API key placeholder with your OpenWeatherMap API key.
4. Run the website in any modern browser.

## How to Use

1. Create an account or log in.
2. Search for your state.
3. Select a city from the suggested list or search for another city.
4. View weather details including:

   * Temperature
   * Humidity
   * Rain Prediction
   * Daily High & Low Temperature
5. Use the Home button to search another city or Logout when finished.
